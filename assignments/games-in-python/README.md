
# 🎮 Assignment: Hangman Game Challenge

## 🎯 Objective

Students will build the classic word‑guessing Hangman game using Python strings, loops, conditionals, and user input. This teaches control flow and basic game logic.

## 📝 Tasks

### 🛠️  Implement game mechanics

#### Description
Write a Python program that runs a text‑based Hangman game. The program should manage word selection, user guesses, display progress, and determine win/lose conditions.

#### Requirements
Completed program should:

- Randomly select a secret word from a predefined list (at least 10 words)
- Prompt the player to guess one letter at a time and read input from the console
- Show the current progress with underscores and revealed letters (e.g. `_ A _ _ M A N`)
- Keep track of incorrect guesses and limit the number of attempts (e.g. 6 wrong letters)
- End the game when the word is fully guessed or attempts run out
- Display a clear win or lose message and reveal the secret word

You may add helpful prompts or visuals, but the core mechanics above must work correctly.
